# Ajay Kumar Portfolio Website

This is a personal portfolio website created using basic HTML, CSS, and JavaScript. The content is based on Ajay Kumar's resume and is arranged in the same order as the resume sections.

## Project Overview

The website presents personal, academic, and technical information in a clean and responsive portfolio format. It includes a profile summary, technical skills, projects, internships, education, certifications, achievements, and contact details.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Responsive design for desktop and mobile screens
- Resume-based portfolio content
- Dark and light theme toggle
- Mobile navigation menu
- Animated statistics
- Animated skill progress bars
- Project filtering by category
- Project details popup
- Contact form validation
- Back-to-top button
- Active navigation highlighting while scrolling

## How to Run

Open the `index.html` file directly in any web browser.

You can also run it with a local server:

```bash
python -m http.server 8000
```

Then open this URL in your browser:

```text
http://127.0.0.1:8000
```
